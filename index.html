Express-React Starter

How To Get Started?
You Must Have:
  • Node Js 8x
  • create-react-app npm package.....To install run npm install -g create-react-app
  • Any Code Editor
Basic Steps
  1. Download a Github repo Here and extract files to your favourite directory.
  2. Install express-server dependencies using npm install command
  3. Install react-client dependencies using npm run client-install command
  4. To run express-server and react-client concurrently use npm run dev command
  5. To run express-server only use npm server command
  6. To run express-server only use npm run client command
  7. express-server runs on http://localhost:5000 and react-client on http://localhost:3000
  8. You are good to go...Happy Coding!
NB: Write your SASS code in src/scss directory
